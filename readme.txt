Contract manager for freelance workers where users can save contract information and upload invoice files.

Users can create and save their own personal profile. Add a contract to a list to have an overview of your work. Notch will then display the average commission and avergae number of days spent on a single contract.
Finally, users can upload invoices to a specific contract to make their important information easy to access and organized.

CRUD webapp using; frontend: HTML, CSS, Bootstrap, Javascript; backend: Python RESTful API on the Flask framework using Jinja templating.
